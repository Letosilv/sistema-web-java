# Sistema Web em Java

Este é um sistema web desenvolvido com Java, PostgreSQL e Docker, incluindo testes end-to-end com Robot Framework.

## 🚀 Tecnologias Utilizadas

- Java 17+
- Spring Boot
- PostgreSQL
- Docker e Docker Compose
- Robot Framework (testes E2E)
- Maven (build)
- Git e GitHub
